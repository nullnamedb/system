Main entry point for all operations.
curl https://yourdb.com/q?q=score=100

Web-based administration interface.
https://yourdb.com/dashboard

Simple Operations

key=valueWrite
Store a key-value pair.
/q?q=score=100
keyRead
Retrieve a value by key.
/q?q=score
key=Delete
Delete a key-value pair.
/q?q=score=

Database Operations

create.databasenameWrite
Create a new database.
/q?q=create.shop
get.Read
List all databases.
/q?q=get.
get.databasenameRead
Get all tables and data from a database.
/q?q=get.shop
delete.databasenameDelete
Delete an entire database.
/q?q=delete.shop

CRUD Operations

add.db.table.col.valueCreate
Add a new record.
/q?q=add.mydb.users.name.John
add.db.table.col.val1,val2,val3Create Multiple
Add multiple records at once.
/q?q=add.mydb.users.name.John,Mary,Tom
get.db.table.idRead by ID
Get a specific record by ID.
/q?q=get.mydb.users.1
update.db.table.id.col=valueUpdate
Update a specific record.
/q?q=update.mydb.users.1.name=Johnny
delete.db.table.idDelete by ID
Delete a specific record.
/q?q=delete.mydb.users.1

Filter & Search

db.table[column=value]
Filter records by column value.
/q?q=mydb.users[age=25]
[column>value]
Greater than filter.
/q?q=mydb.users[age>18]
[column<value]
Less than filter.
/q?q=mydb.users[age<30]
[column~keyword]
Contains / Like filter.
/q?q=mydb.users[name~Jo]
search.keyword
Global search across all databases.
/q?q=search.John

File Operations

add.db.table.col=URL.upload
Download and store file from URL.
/q?q=add.mydb.users.avatar=https://image.jpg.upload
files.list
List all uploaded files.
/q?q=files.list
files.delete.filename
Delete a file.
/q?q=files.delete.photo.jpg

Version Control

commit "message"
Create a commit snapshot.
/q?q=commit "first version"
commits
View commit history.
/q?q=commits
checkout.commit_id
Restore to a specific commit.
/q?q=checkout.commit_123
branch.name
Create a branch.
/q?q=branch.feature
switch.branch
Switch branch.
/q?q=switch.feature
merge.source.into.target
Merge branches.
/q?q=merge.feature.into.main
undo
Undo last operation.
/q?q=undo
redo
Redo last undone.
/q?q=redo
f1, f2, f3
Emergency force back 1-3 commits.
/f1?ses=admin_session

User & Authentication

signup.username.password
Create new user account.
/q?q=signup.john.doe123
login.username.password
Login and get session key.
/q?q=login.john.doe123
logout&ses=key
Logout and invalidate session.
/q?q=logout&ses=abc123
user.listAdmin
List all users.
/q?q=user.list&ses=admin_session
user.role.username.newroleAdmin
Change user role.
/q?q=user.role.john.admin

Backup & Restore

backup

/q?q=backup
backups

/q?q=backups
restore.name
Restore from backup.
/q?q=restore.backup_20250101

Shortcuts & Aliases

a = add, g = get, u = update, d = delete
Shortcuts for CRUD operations.
/q?q=a.mydb.users.name.John
c = create, cm = commit, co = checkout
/q?q=c.mydb
un = undo, re = redo
/q?q=un

Response Formats

&format=json
JSON format (default).
&format=text
Plain text format.
&format=table
ASCII table format.
&format=csv
CSV format for Excel.
&pretty=true
Pretty print JSON.
