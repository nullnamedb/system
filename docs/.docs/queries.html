Basics

https://yourdb.com/q?q=command Public
The main entry point for all database operations. All queries are sent via the q= parameter.
# Example curl https://yourdb.com/q?q=score=100
/dashboard Public
Web-based database administration interface.
https://yourdb.com/dashboard

Data Types

String
Text values enclosed in quotes or plain text.
q=name=John q=message=Hello%20World
Number
Integer or floating point numbers.
q=score=100 q=price=19.99
Boolean
True or false values.
q=active=true q=verified=false
Array
Comma-separated values or JSON array format.
q=names=John,Mary,Tom q=scores=[100,200,300]
Object / JSON
JSON object for complex data structures.
q=user={"name":"John","age":25}
Null
Empty or null value.
q=data=null

Simple Operations (Key-Value)

key=value Write
Store a simple key-value pair.
# Store score /q?q=score=100 # Store name /q?q=name=John
key Read
Retrieve a value by key.
# Get score /q?q=score # Returns: 100
key= Delete
Delete a key-value pair.
# Delete score /q?q=score=

Database Operations

create.databasename Write
Create a new database.
# Create database 'shop' /q?q=create.shop # Create database 'users' /q?q=create.users

Rules: Letters, numbers, underscores only. Max 50 characters.

get. Read
List all databases.
# List all databases /q?q=get.
get.databasename Read
Get all tables and data from a database.
# Get all data from 'shop' /q?q=get.shop
delete.databasename Delete
Delete an entire database (irreversible).
# Delete 'shop' database /q?q=delete.shop
Warning: This action cannot be undone. All data in the database will be permanently deleted.

Table Operations

create.databasename.tablename Write
Create a new table in a database.
# Create 'products' table in 'shop' database /q?q=create.shop.products
create.databasename.tablename.col1,col2,col3 Write
Create a table with predefined columns.
# Create 'users' table with columns /q?q=create.mydb.users.name,email,age
get.databasename.tablename Read
Get all records from a table.
# Get all users /q?q=get.mydb.users
get.databasename.tablename.column Read
Get a specific column from a table.
# Get all names from users table /q?q=get.mydb.users.name
delete.databasename.tablename Delete
Delete a table (irreversible).
# Delete users table /q?q=delete.mydb.users

CRUD Operations (Create, Read, Update, Delete)

add.databasename.tablename.column.value Create
Add a new record to a table.
# Add a user /q?q=add.mydb.users.name.John # Add with ID auto-increment /q?q=add.mydb.users.name.John&age.25
add.databasename.tablename.column.value1,value2,value3 Create Multiple
Add multiple records at once.
# Add multiple users /q?q=add.mydb.users.name.John,Mary,Tom /q?q=add.mydb.users.age.25,30,35
get.databasename.tablename.id Read by ID
Get a specific record by its auto-increment ID.
# Get user with ID 1 /q?q=get.mydb.users.1
update.databasename.tablename.id.column=value Update
Update a specific record.
# Update user name /q?q=update.mydb.users.1.name=Johnny # Update multiple columns /q?q=update.mydb.users.1.name=Johnny&age=26
delete.databasename.tablename.id Delete by ID
Delete a specific record by ID.
# Delete user with ID 1 /q?q=delete.mydb.users.1

Query Operations (Filter & Search)

databasename.tablename[column=value] Filter
Filter records by column value.
# Find users with age 25 /q?q=mydb.users[age=25] # Get name of user with age 25 /q?q=mydb.users[age=25].name
[column>value] Greater Than
Filter records where column is greater than value.
# Users older than 18 /q?q=mydb.users[age>18]
[column<value] Less Than
Filter records where column is less than value.
# Users younger than 30 /q?q=mydb.users[age<30]
[column~keyword] Contains / Like
Filter records where column contains keyword.
# Users with 'Jo' in name /q?q=mydb.users[name~Jo]
[column=value1,value2] In Array
Filter records where column matches any value in list.
# Users with age 25 or 30 /q?q=mydb.users[age=25,30]
search.keyword Global Search
Search across all databases and tables.
# Search for 'John' everywhere /q?q=search.John

File Operations

add.db.table.column=URL.upload Upload File
Download a file from URL and store it locally.
# Upload image from URL /q?q=add.mydb.users.avatar=https://example.com/photo.jpg.upload
files.list List Files
List all uploaded files.
# List all files /q?q=files.list
files.info.filename File Info
Get information about a specific file.
# Get file info /q?q=files.info.photo.jpg
files.delete.filename Delete File
Delete a file from storage.
# Delete file /q?q=files.delete.photo.jpg

Version Control (Git-like)

commit "message" Commit
Create a commit snapshot of current state.
# Create commit /q?q=commit "first version" /q?q=commit "added users table"
commits View History
View commit history.
# List all commits /q?q=commits
checkout.commit_id Checkout
Restore state to a specific commit.
# Restore to commit_123 /q?q=checkout.commit_123
branch.name Create Branch
Create a new branch.
# Create feature branch /q?q=branch.feature-login # List branches /q?q=branches
switch.branch Switch Branch
Switch to a different branch.
# Switch to feature branch /q?q=switch.feature-login
merge.source.into.target Merge
Merge a branch into another.
# Merge feature into main /q?q=merge.feature-login.into.main
undo Undo
Undo the last operation.
# Undo last change /q?q=undo # Undo 3 steps /q?q=undo.3
redo Redo
Redo the last undone operation.
# Redo last undone /q?q=redo
diff.source.target Compare
Compare two commits or branches.
# Compare commits /q?q=diff.commit_123.commit_456 # Compare branches /q?q=diff.main.feature-login

User & Authentication

signup.username.password Sign Up
Create a new user account.
# Create user 'john' /q?q=signup.john.doe123
login.username.password Login
Authenticate and receive session key.
# Login /q?q=login.john.doe123 # Returns: {"success":true,"session":"abc123..."}
logout&ses=session_key Logout
Invalidate current session.
# Logout /q?q=logout&ses=abc123
user.changepass.old.new Change Password
Change your own password.
# Change password /q?q=user.changepass.oldpass.newpass&ses=session
user.list Admin Only
List all users (admin only).
# List users /q?q=user.list&ses=admin_session
user.role.username.newrole Admin Only
Change user role (admin only).
# Make John an editor /q?q=user.role.john.editor&ses=admin_session

Available roles: root, admin, editor, user, viewer

user.delete.username Admin Only
Delete a user (admin only).
# Delete user John /q?q=user.delete.john&ses=admin_session

Backup & Restore

backup Backup
Create a backup of the entire database.
# Create backup /q?q=backup&ses=admin_session # Named backup /q?q=backup.my_backup_name&ses=admin_session
backups List Backups
List all available backups.
# List backups /q?q=backups&ses=admin_session
restore.name Restore
Restore database from a backup.
# Restore from backup /q?q=restore.backup_20250101&ses=admin_session
Warning: Restore will overwrite current data!

System Commands

status Status
Get system status and information.
# Check status /q?q=status
stats Statistics
Get database statistics.
# Get stats /q?q=stats
track Tracking
View query history and tracking logs.
# View last 50 queries /q?q=track # View errors only /q?q=track.errors # View last 24 hours /q?q=track.24hr
help Help
Show help reference.
# Get help /q?q=help

Advanced Features

force.back.steps Force Recovery
Emergency force back N commits.
# Force back 1 commit /q?q=force.back.1&ses=admin_session # Force back 3 commits /q?q=force.back.3&ses=admin_session
f1, f2, f3 Shortcut Force
Emergency force back shortcuts.
# Force back 1 commit /f1?ses=admin_session # Force back 2 commits /f2?ses=admin_session # Force back 3 commits /f3?ses=admin_session
auto.1-9 Auto IDs
Auto-generate sequential IDs.
# Auto IDs 1-9 /q?q=auto.1-9 # With prefix /q?q=auto.255.0001-9999
auto.sortby.name Sorted IDs
Generate IDs sorted by data.
# Sort by name /q?q=auto.1-9.sortby.name.az # Sort by age /q?q=auto.1-9.sortby.age.asc

Shortcuts & Aliases

a = add Alias
Shortcut for add command.
# Same as add.mydb.users.name.John /q?q=a.mydb.users.name.John
g = get Alias
Shortcut for get command.
# Same as get.mydb.users /q?q=g.mydb.users
u = update Alias
Shortcut for update command.
# Same as update.mydb.users.1.name=Johnny /q?q=u.mydb.users.1.name=Johnny
d = delete Alias
Shortcut for delete command.
# Same as delete.mydb.users.1 /q?q=d.mydb.users.1
c = create, cm = commit, co = checkout Aliases
Shortcuts for create, commit, checkout.
/q?q=c.mydb /q?q=cm "message" /q?q=co.commit_123
un = undo, re = redo Aliases
Shortcuts for undo and redo.
/q?q=un /q?q=re

Response Formats

&format=json
JSON format (default).
/q?q=get.users&format=json
&format=text
Plain text format for terminals.
/q?q=get.users&format=text
&format=table
ASCII table format.
/q?q=get.users&format=table
&format=csv
CSV format for Excel.
/q?q=get.users&format=csv
&pretty=true
Pretty print JSON.
/q?q=get.users&pretty=true
&raw=true
Raw output without metadata.
/q?q=get.users&format=text&raw=true
