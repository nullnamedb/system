NullName DB API Reference

RESTful API — Simple, Powerful, Free

Overview

Base URL: https://yourdb.com/

https://yourdb.com/q?q=your_command_here

Authentication: Session key via &ses=key or X-Session header.
Response Formats: JSON (default), CSV, Text, Table

Authentication

POST/q?q=login.username.password
curl https://yourdb.com/q?q=login.admin.nullname2025
POST/q?q=signup.username.password
curl https://yourdb.com/q?q=signup.john.doe123

Data Operations

POST/q?q=create.databasename
curl https://yourdb.com/q?q=create.mydb
POST/q?q=add.db.table.column.value
curl https://yourdb.com/q?q=add.mydb.users.name.John
GET/q?q=get.db.table
curl https://yourdb.com/q?q=get.mydb.users
GET/q?q=get.db.table.id
curl https://yourdb.com/q?q=get.mydb.users.1
PUT/q?q=update.db.table.id.column=value
curl https://yourdb.com/q?q=update.mydb.users.1.name=Johnny
DELETE/q?q=delete.db.table.id
curl https://yourdb.com/q?q=delete.mydb.users.1

File Operations

POST/upload
curl -X POST -F "file=@/path/to/image.jpg" https://yourdb.com/upload?ses=key
POST/upload/url
curl -X POST -H "Content-Type: application/json" -d '{"url":"https://example.com/image.jpg"}' https://yourdb.com/upload/url?ses=key

Version Control

POST/q?q=commit "message"
curl https://yourdb.com/q?q=commit "first version"
GET/q?q=commits
POST/q?q=branch.name
curl https://yourdb.com/q?q=branch.test
POST/q?q=merge.source.into.target
curl https://yourdb.com/q?q=merge.test.into.main
POST/q?q=undo
GET/f1, /f2, /f3Admin
curl https://yourdb.com/f1?ses=admin_session

User Management

GET/q?q=user.listAdmin
POST/q?q=user.role.username.newroleAdmin
DELETE/q?q=user.delete.usernameAdmin

System

GET/health
{"status":"ok","timestamp":"..."}
GET/q?q=status
GET/q?q=stats
GET/q?q=help

Response Formats

# CSV
curl https://yourdb.com/q?q=get.users&format=csv

# Plain Text
curl https://yourdb.com/q?q=get.users&format=text

# Table
curl https://yourdb.com/q?q=get.users&format=table

# Pretty JSON
curl https://yourdb.com/q?q=get.users&pretty=true

Error Codes

CodeMeaning
401Unauthorized - Authentication required
403Forbidden - Insufficient permissions
404Not Found
429Too Many Requests
500Internal Server Error

Code Examples

JavaScript

const res = await fetch('https://yourdb.com/q?q=score=100');
const data = await res.json();

Python

import requests
r = requests.get('https://yourdb.com/q?q=score=100')
print(r.json())

cURL

curl "https://yourdb.com/q?q=add.mydb.users.name.John&ses=abc123"
