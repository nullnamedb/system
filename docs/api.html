Base URL

https://yourdb.com/

Overview

NullName DB provides a simple HTTP API for all database operations. All endpoints accept GET and POST requests with the query string parameter q=.

https://yourdb.com/q?q=your_command_here

Base URL: https://yourdb.com/
Response Format: JSON (default), CSV, Text, Table
Authentication: Session key via &ses=key or X-Session header

Authentication

POST /q?q=login.username.password

Authenticate and receive a session key.

curl https://yourdb.com/q?q=login.admin.nullname2025
Response: 
{"success":true,"session":"abc123...","user":{"username":"admin","role":"admin"}}
POST /q?q=signup.username.password

Create a new user account.

curl https://yourdb.com/q?q=signup.john.doe123
GET /q?q=logout&ses=session_key

Invalidate current session.

Data Operations

Simple Key-Value

GET/POST /q?q=key=value Public

Store a value.

curl https://yourdb.com/q?q=score=100
GET /q?q=key Public

Retrieve a value.

curl https://yourdb.com/q?q=score

Structured Data

POST /q?q=create.databasename

Create a new database.

curl https://yourdb.com/q?q=create.mydb
POST /q?q=add.db.table.column.value

Add data to a table.

curl https://yourdb.com/q?q=add.mydb.users.name.John
ParameterDescription
dbDatabase name
tableTable name
columnColumn name
valueValue to store
GET /q?q=get.db.table

Retrieve all data from a table.

curl https://yourdb.com/q?q=get.mydb.users
GET /q?q=get.db.table.id

Retrieve a specific record by ID.

curl https://yourdb.com/q?q=get.mydb.users.1
PUT /q?q=update.db.table.id.column=value

Update a specific record.

curl https://yourdb.com/q?q=update.mydb.users.1.name=Johnny
DELETE /q?q=delete.db.table.id

Delete a specific record.

curl https://yourdb.com/q?q=delete.mydb.users.1

File Operations

POST /upload

Upload a file (multipart/form-data).

curl -X POST -F "file=@/path/to/image.jpg" https://yourdb.com/upload&ses=session_key
POST /upload/url

Upload a file from a URL.

curl -X POST -H "Content-Type: application/json" -d '{"url":"https://example.com/image.jpg"}' https://yourdb.com/upload/url?ses=session_key
GET /files/{filename}

Download a file.

curl https://yourdb.com/files/image.jpg
GET /q?q=files.list

List all uploaded files.

Version Control

POST /q?q=commit "message"

Create a commit snapshot.

curl https://yourdb.com/q?q=commit "first version"
GET /q?q=commits

List commit history.

POST /q?q=branch.name

Create a new branch.

curl https://yourdb.com/q?q=branch.test
POST /q?q=merge.source.into.target

Merge branches.

curl https://yourdb.com/q?q=merge.test.into.main
POST /q?q=undo Auth Required

Undo last operation.

GET /f1, /f2, /f3 Admin Only

Emergency force back 1, 2, or 3 commits.

curl https://yourdb.com/f1?ses=admin_session

Backup & Restore

POST /q?q=backup

Create a backup.

curl https://yourdb.com/q?q=backup&ses=admin_session
GET /q?q=backups

List all backups.

POST /q?q=restore.name

Restore from backup.

curl https://yourdb.com/q?q=restore.backup_20250101&ses=admin_session

User Management

GET /q?q=user.list Admin Only

List all users.

POST /q?q=user.role.username.newrole Admin Only

Change user role.

curl https://yourdb.com/q?q=user.role.john.editor&ses=admin_session
DELETE /q?q=user.delete.username Admin Only

Delete a user.

POST /q?q=user.changepass.old.new

Change your own password.

System Endpoints

GET /health Public

Health check endpoint.

{"status":"ok","timestamp":"2025-01-01T00:00:00.000Z","stats":{...}}
GET /q?q=status

System status information.

GET /q?q=stats

Database statistics.

GET /q?q=help Public

Help reference.

Response Formats

Use the format parameter to specify response format:

FormatParameterContent-TypeExample
JSON&format=jsonapplication/jsonDefault, prettified with &pretty=true
Plain Text&format=texttext/plainHuman readable key:value pairs
Table&format=tabletext/plainASCII table format
CSV&format=csvtext/csvComma-separated values for Excel
JSONP&format=jsonp&callback=funcapplication/javascriptJSON with padding
# Get as CSV
curl https://yourdb.com/q?q=get.users&format=csv

# Get as table
curl https://yourdb.com/q?q=get.users&format=table

# Get pretty JSON
curl https://yourdb.com/q?q=get.users&pretty=true

Code Examples

JavaScript (Fetch)

// Save data
const response = await fetch('https://yourdb.com/q?q=score=100');
const data = await response.json();

// Get data
const getResponse = await fetch('https://yourdb.com/q?q=score');
const value = await getResponse.json();

Python (Requests)

import requests

# Save data
response = requests.get('https://yourdb.com/q?q=score=100')

# Get data
response = requests.get('https://yourdb.com/q?q=score')
print(response.json())

cURL

# Save
curl "https://yourdb.com/q?q=score=100"

# Get
curl "https://yourdb.com/q?q=score"

# With session
curl "https://yourdb.com/q?q=add.mydb.users.name.John&ses=abc123"

# With format
curl "https://yourdb.com/q?q=get.users&format=table"

Node.js

const https = require('https');

const query = 'score=100';
https.get(`https://yourdb.com/q?q=${encodeURIComponent(query)}`, (res) => {
    let data = '';
    res.on('data', chunk => data += chunk);
    res.on('end', () => console.log(JSON.parse(data)));
});

PHP

Go

import (
    "fmt"
    "io/ioutil"

    body, _ := ioutil.ReadAll(resp.Body)

}
