Getting Started

NullName DB is a zero-configuration, file-based database. No installation, no setup, no payment.

# Save data
/q?q=score=100

# Retrieve data
/q?q=score
# Returns: 100
Tip: Access the dashboard at /dashboard for a visual interface.

Users & Roles

Role-based access control with simple username/password authentication. No email required.

RolePermissions
ADMINEverything (read, write, delete, manage users)
USERRead + write own data
VIEWERRead only
 
# Signup (username + password only)
/q?q=signup.john.doe123

# Login (returns session key)
/q?q=login.john.doe123

# All queries need session
/q?q=add.mydb.users.name.John&ses=abc123

Queries

# Create database
/q?q=create.mydb

# Add data
/q?q=add.mydb.users.name.John
/q?q=add.mydb.users.age.25

# Get data
/q?q=get.mydb.users

# Update
/q?q=update.mydb.users.1.name=Johnny

# Delete
/q?q=delete.mydb.users.1

Response Formats

FormatParameter
JSON&format=json

Version Control

# Create commit
/q?q=commit "initial setup"

# View history
/q?q=commits

# Create branch
/q?q=branch.feature

# Merge
/q?q=merge.feature.into.main

# Undo last change
/q?q=undo

# Force recovery
/f1&ses=admin_session

File Management

# Upload from URL
/q?q=add.mydb.users.avatar=https://image.jpg.upload

# List files
/q?q=files.list

# Delete file
/q?q=files.delete.filename.jpg

Backup & Restore

# Create backup
/q?q=backup

# List backups
/q?q=backups

# Restore
/q?q=restore.backup_name

FAQ

Is it really free?

Yes. Completely free. No premium tiers. No payment. Forever.

Where is my data stored?

As encrypted JSON files in /database/path/ on your VPS.

No email? No password recovery?

Correct. Simple username + password only. No email, no reset. Keep your password safe.

How to update?

Run git pull in installation directory, then sudo systemctl restart nullname.
